# Security Policy

## Supported Versions

Este projeto ainda não mantém matriz formal de versões suportadas.
Relate vulnerabilidades para a versão mais recente disponível no repositório.

## Reporting a Vulnerability

Abra uma issue com o rótulo `security` contendo:

- Descrição do problema
- Impacto esperado
- Passos de reprodução
- Evidências/logs mínimos

Evite publicar segredos, tokens ou dados sensíveis nos anexos.
